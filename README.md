# Professional LLM Chatbot

This project is a professional chatbot web app using Streamlit and OpenAI's GPT models.  
It supports dynamic chat, session state, and is ready for deployment/sharing.

## Usage

1. Clone the repo
2. Install dependencies:  
   `pip install -r requirements.txt`
3. Run the app:  
   `streamlit run app.py`
4. Enter your OpenAI API key in the sidebar and start chatting!

## Features

- Python script generation
- Step-by-step explanations
- Professional tone and session context
- Easy for extension to any topic/domain
